## Para que sirve MarkDown:

- Sirve para mostrar textos de forma sencila, con una sintaxis más simple que la de HTML.
- Es utilizado principalmente para documentación.
- '#' Sirve para definir headers, puede llegar hasta varios niveles.
- Encerrar palabras entre '\*\*' las muestra en negritas.
- Hay dos tipos de listas, ordenada y no ordenada. La no ordenada se escribe iniciando con '-' y la ordenada con '1.'
- Las comillas al reves ' ``` ' son utilizadas para mostrar codigo generalmente.
